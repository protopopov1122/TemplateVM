## Compile-time VM

It's one-evening project that aims to implement a simple stack-based virtual machine in C++ which runs completely in compile-time. Currently it implements common VM infrastructure, push and 5 arithmetical commands. I'm planning to add some more 
arithmetical-logical and flow control commands. The main reason of this project - curiosity to write entirely compile-time virtual machine. All project code is licensed under the terms of WTFPL. \
